Students on Rails 2015 Cheatsheet

shell basics

mkdir <dir> Verzeichnis <dir> erstellen
ls <dir> Inhalte in dem Verzeichnis <dir> anzeigen
cd <dir> Nach <dir> wechseln
mv <foo> <dir/bar> Verzeichnis oder Datei <foo> in das Verzeichnis <dir> verschieben und in <bar> umbenennen
rm <file> <file> löschen
rm -r <dir> <dir> und alles darin löschen (-r bedeutet recursive)
rmdir <dir> Leeres Verzeichnis <dir> löschen

rails & rake

rails new <name> Ein neues Rails Projekt anlegen
rails generate migration <name> Datenbank-Migration erstellen
rails generate model <name> Model erstellen
rake db:create Datenbank erstellen
rake db:migrate Datenbank-Migrationen ausführen

git

git init Ein neues lokales Git Repository anlegen
git clone Ein bestehendes Repository von einem remote Server clonen
git status Aktuellen Status von geänderten und/oder hinzugefügenten Dateien im Repository anzeigen
git add <file*> Datei(en)/Änderungen für den nächsten Commit vormerken
git commit Vorgemerkte Datei(en)/Änderungen in das lokale Repostitory schreiben
git pull <server> <branch> Commits von einem remote Server herunterladen
git branch <name> Einen neuen Branch namens <name> erstellen
git checkout <name> Den Branch <name> auschecken
git push <server> <branch> Commits auf dem aktuellen lokalen Branch auf den remote Branch <branch> hochladen

arrays

tiere = ["hund", "katze", "maus"] Erstellen
tiere[0] #=> "hund" Zugriff
tiere[1] = "tiger" Zuweisung
tiere.length #=> 3 Größe eines Arrays
tiere << "giraffe" Element hinzufügen
tiere.each { |tier| puts tier } Iteration

hashes

person = { :name => "Hans" } Erstellen
person = { name: "Hans" } Alternative Syntax für Erstellung
person[:name] #=> "Hans" Zugriff
person[:name] = "Fritz" Zuweisung

Iteratoren

# Mit each eine Aktion für jedes Element einer Sammlung ausführen

[1, 2, 3].each { |zahl| puts zahl+1 }

[1, 2, 3].each do |zahl| puts zahl+1 end

Klassen

class Person def initialize(name) @name = name end
def vorstellen puts "Hallo, ich bin #{@name}." end end

Getter und Setter

class Person attr_reader :name # Lesen des Attributs attr_writer :name # Schreiben des Attributs attr_accessor :name # Lesen und Schreiben des Attributs end

Vererbung

class Softwareentwickler < Person def vorstellen super puts "Ich bin Softwareentwickler!" end end

Module

# nicht instanziierbar
# Methoden können von Klassen übernommen werden

module MeinModul def test puts "Dies ist ein Test." end end

class MeineKlasse include MeinModul end

MeineKlasse.new.test

# Schutz vor Namenskollisionen:

module MeinModul class MeineKlasse def test puts "Dies ist ein weiterer Test." end end end

MeinModul::MeineKlasse.new.test
# Ausgabe: "Dies ist ein weiterer Test."

# besonderes Modul: Enumerable (führt alle Methoden auf each zurück)

[1, 2, 3].all? { |zahl| zahl > 0 }
[1, 2, 3].map { |zahl| zahl*-1 }
